React app to fetch a a list of random [Wikipedia](https://www.wikipedia.org/) articles.

:book: :eyes: :bulb:

#### Install
```
git clone git@github.com:jonoliver/wikirandom.git
yarn install
```

#### Run
```
yarn run
```

#### Test
```
yarn test
```

#### Build
```
yarn build
```

#### Deploy
```
yarn deploy
```
