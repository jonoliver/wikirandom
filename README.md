React app to fetch a list of random [Wikipedia](https://www.wikipedia.org/) articles. [Try it out!](https://jonoliver.github.io/wikirandom)

:book: :eyes: :bulb:

#### Install
```
git clone git@github.com:jonoliver/wikirandom.git
yarn install
```

#### Run
```
yarn run
```

#### Test
```
yarn test
```

#### Build
```
yarn build
```

#### Deploy
```
yarn deploy
```
